1. 대여소 등록
actor : 관리자
requirement : 관리자는 새로운 대여소 등록 가능(대여소 이름, 위치, 자전거 보관 가능 수량, 운영 시간)

2. 대여소 조회
actor : 관리자
requirement : 관리자는 전체 대여소 목록 조회 가능, 선택 시 상세정보 열람 가능

3. 대여소 삭제
actor : 관리자
requirement : 관리자는 기존 대여소 삭제 가능

4. 자전거 등록
actor : 관리자
requirment : 관리자는 자전거를 시스템에 등록 가능(자전거 ID, 제품명, 유형, 소속 대여소, 상태)

5. 자전거 조회
actor : 관리자
requirement : 관리자는 등록된 자전거 조회 가능. 특정 자전거 항목을 선택하여 상세 내용 확인 가능

6. 자전거 삭제
actor : 관리자
requirement : 관리자는 등록된 자전거 리스트 조회 화면에서 특정 자전거 삭제 가능

7. 대여소 검색
actor : 회원
requirement : 회원은 대여소의 이름을 검색하여 출력된 대여소 리스트 확인 가능

8. 자전거 대여
actor : 회원
requirement : 대여소 리스트에서 자전거 대여 가능
